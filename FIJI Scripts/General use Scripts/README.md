# FIJI_Scripts 

## Save all opened images.ijm
Exactly that into a selected folder

## DrawFeret.ijm
Draws the Feret of a ROI

## DrawEllipse.ijm
Draws the best fitting ellipse to all ROIs

## Batch processing template using defined functions.ijm
Process folder and subfolders with images using functions

## Batch processing template using defined functions.ijm
Process folder of stacks, measuring on every frame and creating a dataframe in the Log windows. Finally it draws results on the original image

## Fixed Length Line Tool.ijm
This ImageJ tool macro allows for drawing a line with a given fixed length or maximum length.

## pruning_skeleton Threshold
Imagej Macro that cleans/prunes branches of a skeleton under a length threshold
